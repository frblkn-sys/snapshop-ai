# SnapShop AI 🚀

AI-powered e-commerce assistant developed for BTK Hackathon 2026.

## Problem
Online sellers struggle to calculate platform fees, profits, and product optimization quickly.

## Solution
SnapShop AI analyzes product images and helps sellers estimate fees, pricing, and profitability using AI.

## Features
- Product image analysis
- AI-generated product insights
- Platform fee calculation
- Profit estimation
- Streamlit dashboard

## Technologies
- Python 
- Streamlit
- Gemini API
- Pandas

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
